Localized Images

A truly localized documentation site needs the screenshots to be localized as well. At this time, and as far as I can tell, Docusaurus doesn’t address this need directly.

Our first thought was to have translators add localized screenshots using Crowdin’s Assets feature. However, we decided that it would be too hard for translators to be expected to know how to recreate the conditions in each screenshot. It is much less expensive to ask the author of the documentation to create the various screenshot. This can be done either at the same time as the English ones are created, or later.

This then opens up the option of not using crowdin, but instead embedding localized images, somehow, in the notion version of the page. We are experimenting with that, here:
